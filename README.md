# Motivational Quotes Telegram Bot

This is a simple Telegram bot created for the Introduction to Python Web Development course.

## Project Description

The bot sends motivational quotes to users. It can also send quotes based on the user's mood and save favorite quotes.

## Features

- /start - welcome message
- /help - list of commands
- /quote - random motivational quote
- /mood happy - quote for happy mood
- /mood sad - quote for sad mood
- /mood tired - quote for tired mood
- /mood stressed - quote for stress
- /favorite - save the last quote
- /favorites - show favorite quotes
- /add - add your own quote
- /myquotes - show your own quotes
- /about - project information

## Technologies Used

- Python 3
- Telegram Bot API
- python-telegram-bot
- JSON file storage
- VS Code
